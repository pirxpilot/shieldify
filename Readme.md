# shieldify

Replace .png badges with .svg badges from [shields.io]

## Installation

```
npm install -g shieldify
```

## Usage

```
shieldify   # will fix/replace badges in Readme.md
```

or if your Readme file is not called `Readme.md`


```
shieldify /path/to/README
```
